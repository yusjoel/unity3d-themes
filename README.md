# Unity Forum
unity.com域名虽然没有被封掉, 但是引用了一些unity3d.com下的资源.

这里做一个镜像, 可以使用[Gooreplacer](https://github.com/jiacai2050/gooreplacer)之类的工具进行重定向

论坛内部使用Sajari进行搜索和转向, 这也是一个不科学的网站.

2. 匹配模式: unity3d.com/profiles/unity3d/themes/
